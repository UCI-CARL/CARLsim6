6 ### Quickstart
7 
8 The current version of the CARLsim paramter-tuning framework uses Evolutionary
9 Computations in Java (ECJ) (version 22 or greater is required). For information
10 on how to install ECJ, please go [here](http://cs.gmu.edu/~eclab/projects/ecj/).
11 
12 
13 
14 1) Set the ECJ_JAR and ECJ_PTI_DIR variables in ~/.bashrc.
15  NOTE: the ECJ_JAR includes the name of the ECJ jar file.
16  EXAMPLE: if the ECJ jar file is found in /opt/ecj/jar, and named
17  ecj.22.jar, then:
18  ECJ_JAR=/opt/ecj/jar/ecj.22.jar.
19 
20 2) Change the current directory to ’tools/ecj_pti’.
21 
22 3) Type ‘make clean && make && make install’
23 
24 4) Refer to http://uci-carl.github.io/CARLsim4/ch10_ecj.html and http://uci-carl.github.io/CARLsim4/tut7_pti.html for installation and
25  how to use CARLsim and ECJ to tune SNNs.
26 
27 TO UNINSTALL:
28 CARLsim: Remove the folder where you installed the CARLsim ECJ PTI library. This folder is located in $(ECJ_PTI_DIR).
29 
30 Type ‘make help’ for additional information.

72 ### Using the CARLsim testing framework
73 
74 CARLsim uses the googletest framework v1.7 for testing. For more information
75 on googltest please visit the website at https://code.google.com/p/googletest/.
76 For a quick primer, visit: https://code.google.com/p/googletest/wiki/Primer.
77 
78 To use the googletest framework, you must first download googletests and
79 point to the googletest directory. One must link to the correct library (either
80 gtest.a or gtest_main.a) and include the correct headers during compilation.
81 Please see the test/CARLsim-app/Makefile for examples of how to compile your
82 tests.
